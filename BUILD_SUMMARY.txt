HOW TO GET STARTED
================================================================================

1. SETUP (5 minutes with Docker)
   $ git clone <repo>
   $ cd grindstone-apex
   $ cp .env.example .env
   $ docker-compose up -d

2. VERIFY (1 minute)
   $ curl http://localhost:8001/health
   $ open http://localhost:8001/docs

3. TEST (5 minutes)
   $ python examples/quick_start.py
   # Runs 5 scenarios showing full workflow

4. GENERATE STRATEGIES
   $ curl -X POST http://localhost:8001/api/v1/generate/initial \
     -d '{"pair": "BTC/USDT", "count": 100}'

5. BACKTEST
   $ curl -X POST http://localhost:8001/api/v1/backtest/batch \
     -d '[{strategy1}, {strategy2}, ...]'

6. MONITOR
   $ curl http://localhost:8001/api/v1/ralph-loop/elite
   $ curl http://localhost:8001/api/v1/system/metrics

Total setup time: 30-45 minutes (Docker) or 1-2 hours (local)
